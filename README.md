### Install Jovo-Inbox


### Install services
Run setup (installs dependecies for `/core` `/api` and `/frontend`)

```
$ npm run setup

or

$ npm run setup:core
$ npm run setup:api
$ npm run setup:frontend
```

### Start services

#### API service
```
$ cd api
$ npm run start:dev
```

#### Frontend

```
$ cd frontend
$ npm run serve
```



Open  http://localhost:8080
